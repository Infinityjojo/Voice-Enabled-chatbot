This is a \*\*voice- and text-enabled chatbot\*\* built with \*\*Streamlit\*\*, \*\*NLTK\*\*, and \*\*scikit-learn\*\*.  



It allows users to:



\- Upload or paste text to create a knowledge base.

\- Chat using \*\*text input\*\* or \*\*voice input via uploaded audio files\*\*.

\- Export chat history as a `.txt` file.



---



\## Features



\- Text normalization, tokenization, and lemmatization with \*\*NLTK\*\*.

\- Similarity-based response retrieval using \*\*TF-IDF\*\* and \*\*cosine similarity\*\*.

\- Small talk and greeting recognition.

\- Session-based chat history in Streamlit.

\- Cloud-friendly: Voice input works via uploaded audio files.
